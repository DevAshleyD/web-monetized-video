The <web-monetized-video> component

The <web-monetized-video> is a web component that you can use to hides ads if it determines the user is sending payments via web monetization.

The component only activates if the Web Monetization API is available, otherwise it will just show it's contents.

The component has 2 potential attributes you can use:

timeout
Sets the amount of time, in milliseconds, that the component will wait for web monetization to start sending payments. Default is 3000 (3 seconds).
ad-template
A CSS selector for a <template> element. The component will find the template, hydrate it and use it as the ad contents. This overrides nesting a template element within the custom element.

Examples

If you are using a browser that doesn't support the Web Monetization API then you will immediately see the text "Adverts" below. If your browser supports Web Monetization and you send a payment to the page then the text "Adverts" will either not appear or disappear once you start sending payments.

<web-monetized-video>
  <template>
    <p><strong>Adverts</strong></p>
  </template>
</web-monetized-video>
